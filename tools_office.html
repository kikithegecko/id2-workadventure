Gemeinsames Editieren von Dokumenten

Die niedersächsische Landesbeauftragte für Datenschutz hat in 2019 bekannt gegeben, Microsoft Office 365 auf Vereinbarkeit mit dem geltenden Datenschutz zu prüfen. Bisher ist diese Prüfung noch nicht abgedschlossen, eine Verwendung von 365 ist somit riskant. Doch es gibt datenschutzfreundlichere Alternativen, um gemeinsam Dokumente zu bearbeiten.

OnlyOffice

OnlyOffice ist eine web-basierte vollständige Office-Lösung, die insbesondere kollaboratives Editieren ermöglicht. Eine hohe Kompatibilität mit den Microsoft Office-Dateiformaten wird laut Hersteller gewährleistet. Die Software ist quelloffen, man kann also selbst eine komplett eigene OnlyOffice-Umgebung installieren, oder einen externen Dienstleister für die Bereitstellung beauftragen. Das dahinter stehende Unternehmen kommt aus Estland.

Etherpad und EtherCalc

Wenn es um das gemeinsame Entwerfen von Texten geht, ist der Text-Editor

Etherpad

zu empfehlen. Eine frei zugängliche Installation aus Deutschland ist beispielsweise YoPad, das vom Bundesjugendring betrieben wird. Die quelloffene Software Etherpad Lite kann jedoch auch auf einem eigenen Server installiert werden.

Inspiriert von Etherpad gibt es zudem das Projekt EtherCalc für kollaborative Tabellenkalkulation, die auch eine Import-Funktion beispielsweise für Office-Dokumente besitzt. Unter ethercalc.net kann der Dienst kostenlos ausprobiert werden. Auch EtherCalc ist Open Source Software.
